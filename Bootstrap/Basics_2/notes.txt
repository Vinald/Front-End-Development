# how to setup environment variables locally to use bootstrap

1. install node js 
https://nodejs.org/en

2. create the package folder
 command npm init

3. install packages
 - dart sass

    > https://www.npmjs.com/package/dart-sass

    > command npm install --save-dev sass // for a particular project

    > command npm install -g sass for globally available

 - bootstrap 5
    > https://getbootstrap.com/docs/5.2/getting-started/download/

    > command npm install bootstrap

 - fontawesome
    > https://fontawesome.com/docs/web/setup/packages 

    > command npm install --save @fortawesome/fontawesome-free


 - autoprefixer
    > command npm install postcss-cli autoprefixer --save

- npm script to like sass to css

then after ur start npm 

- customize the bootstrap file
